# Instagram Reuploader

Десктопное приложение под Windows: качает видео из Instagram, видоизменяет (яркость, контраст, насыщенность, скорость, кроп, опц. зеркало + баннер) и сохраняет на диск.

## Как получить готовый .exe (без установки чего-либо)

### Один раз: создать репозиторий

1. Зарегистрируйся на [github.com](https://github.com), если ещё нет.
2. Нажми **+ → New repository**, назови `insta-reuploader` (или как хочешь), сделай **Public** (для приватных репо тоже работает, но публичные дают неограниченный лимит Actions).
3. **Не** ставь галки про README/license — оставь пустой.
4. Нажми **Create repository**.

### Загрузить файлы

Самый простой способ — через веб-интерфейс GitHub:

1. На странице нового репозитория нажми **uploading an existing file** (ссылка прямо в центре).
2. Перетащи туда **все файлы из этой папки**, сохраняя структуру:
   ```
   app.py
   requirements.txt
   README.md
   .github/workflows/build.yml
   ```
3. Внизу нажми **Commit changes**.

> Если перетащить папку `.github` целиком не получается (бывает в Windows-проводнике из-за точки в начале имени), создай файл через веб: **Add file → Create new file** → в поле имени напиши `.github/workflows/build.yml` (слэши автоматически создадут подпапки), вставь содержимое, закоммить.

### Запустить сборку

1. На странице репозитория открой вкладку **Actions**.
2. Если GitHub спросит "Workflows aren't being run on this repository" — нажми **I understand my workflows, go ahead and enable them**.
3. Слева в списке выбери **Build Windows EXE**.
4. Справа жми **Run workflow → Run workflow** (зелёная кнопка).
5. Жди 3–5 минут. Когда галочка станет зелёной — клик на запуск.
6. Внизу страницы будет блок **Artifacts** с файлом **InstaReuploader-Windows**. Скачивай — внутри `InstaReuploader.exe`.

Готово, бери и пользуйся 🎉

### Авто-релизы (опционально)

Если хочешь, чтобы exe лежал на странице **Releases** для удобной раздачи:

1. На главной странице репо справа жми **Releases → Create a new release**.
2. **Choose a tag** → впиши `v1.0` → **Create new tag**.
3. Жми **Publish release**.
4. Через 3–5 минут exe сам прикрепится к релизу (см. вкладку Actions для прогресса).

При следующих изменениях кода: `v1.1`, `v1.2` и т.д.

## Как пользоваться

1. Запусти `InstaReuploader.exe` (первый запуск может занять 5-10 сек — это ffmpeg распаковывается во временную папку).
2. Вставь ссылку на Instagram Reels.
3. Выбери папку, куда сохранять.
4. (Опц.) выбери баннер — PNG с прозрачным фоном.
5. (Опц.) для приватных/защищённых видео подложи `cookies.txt` — экспортируется расширением «Get cookies.txt LOCALLY» из браузера, где ты залогинен в Instagram.
6. Жми **🚀 Обработать**.

Настройки сохраняются в `C:\Users\<Имя>\InstaReuploader\config.json` между запусками.

## Локальный запуск (для разработки)

Если хочешь править код без сборки:

```cmd
pip install -r requirements.txt
python app.py
```

Понадобится ffmpeg в PATH или `ffmpeg.exe` в папке со скриптом.

## Лицензия

Личное использование. Соблюдай правила Instagram и авторские права на исходные видео.
